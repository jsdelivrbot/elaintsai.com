Daily UI #006
User Profile
Design a user profile and be mindful of the most important data, names, imagery, placement, etc. Is it for a serious profile? A social profile?

This mockup focuses on the profile of a customer using the HBX app. The main goal is to provide an easy point of reference for past purchases and favorites. The ‘favorites’ tab is a functionality that currently does not exist on HBX, but I believe it is a great enhancement to increase conversion rates.

This is my first project using Sketch and Principle!
Daily UI #005
Calculator
Design a calculator. Standard, scientific, or specialty calculator for something such as a mortgage? Is it for a phone, a tablet, a web app?

How easy would traveling be if you could calculate and convert currency within the same app? It may be difficult to remember the exchange rate or do the calculations on the spot without a tool, so this currency calulator is the fix to your purchasing worries abroad!

App Icon
Design an app icon. What best represents the brand or product? Or is it incredibly unique? Does it look great at a distance and does it stand out when put on your home screen alongside other apps?

An app icon for a travel budgeting app.

Daily UI #002
Checkout Page
Design a credit card checkout form or page. Don't forget the important elements such as the numbers, dates, security numbers, etc. (It's up to you!)

I designed a mobile checkout page for an airline. My main objective is to make the itinerary stand out enough so that the user fully understands what flight they are booking. I am terrified of booking the wrong flight, so having the information at the top is reassuring.
